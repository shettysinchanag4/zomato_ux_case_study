 # Zomato UX Case Study 🍔📱

A modern and detailed **UX Case Study** of the Zomato food delivery platform, focused on improving user experience through research, journey mapping, SWOT analysis, and redesign solutions.

This project analyzes real user pain points in the food ordering experience and proposes high-impact UI/UX improvements using design thinking principles and usability research.

## ✨ Features

* Modern responsive UI design
* Complete UX audit of the Zomato platform
* User Journey Mapping
* SWOT Analysis
* High-fidelity redesign proposals
* Before vs After wireframe comparisons
* UX design principles implementation
* Clean typography and professional layout
* Mobile responsive design

## 📌 Key UX Improvements Proposed

* Decluttered Home Screen
* Streamlined Single-Screen Checkout
* Contextual & Delayed Rating System

## 🛠️ Technologies Used

* HTML5
* CSS3
* Responsive Web Design
* UI/UX Design Principles

## 🎯 Objectives

* Improve usability and accessibility
* Reduce user friction during ordering
* Enhance checkout experience
* Improve customer engagement and review quality

## 📖 Research Methods

* Heuristic Evaluation
* Competitor Analysis
* User Journey Mapping
* Cognitive Walkthrough
* App Review Research

## 🚀 Project Outcome

This case study demonstrates how thoughtful UX decisions can improve user satisfaction, reduce cart abandonment, and create a smoother food delivery experience.
